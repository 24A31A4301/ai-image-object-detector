# 🧠 AI Image Object Detection System

## 📖 Project Description
This project is an AI-based web application that detects objects in images using a pre-trained deep learning model. The system allows users to upload an image and automatically identifies objects present in the image.

## 🚀 Features
- Upload image from device
- Detect multiple objects
- Display object names with confidence score
- Fast and real-time processing

## 🧠 Technologies Used
- HTML
- JavaScript
- TensorFlow.js
- COCO-SSD Model

## ⚙️ How It Works
1. User uploads an image
2. The AI model processes the image
3. Objects are detected using deep learning
4. Results are displayed with confidence percentage

## 📸 Sample Output
Example:
Person (95%)
Dog (88%)

## 🎯 Applications
- Security systems
- Image recognition
- Smart surveillance
- AI-based automation

## 📌 Conclusion
This project demonstrates how AI and deep learning can be used in web applications for object detection using pre-trained models.
